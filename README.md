# Password Generatore



### First step
```bash
git clone git@github.com:azimhatami/PassGen.git
```


### Change directory
```bash
cd PassGen/
```


### Create venv
```bash
make venv
```


### Activate virtual env
```bash
source activate.sh
```


### Installation module
```bash
make install
```


### Now run
```bash
python3 passgen.py
```
